# Gardner Perceptron

A minimal C++17 library for single-layer perceptron experiments in the teacher-student framework, designed for numerical exploration of the statistical mechanics of learning.

## Theoretical background

The library implements the setting studied analytically by Elizabeth Gardner (1988) and treated at length in *Statistical Mechanics of Learning* by Engels & Van den Broeck. The central object is a single-layer perceptron with ±1 inputs and weights drawn from a Gaussian prior. The key questions — how many patterns can be stored, how well does a trained network generalise — are addressed by the **replica method**, borrowed from the theory of spin glasses (Sherrington-Kirkpatrick model). The replica-symmetric saddle point gives closed-form predictions for the generalisation error ε as a function of α = P/L, the ratio of training examples to input dimension.

The library lets you simulate these predictions directly:

- **Hebb rule** — one-shot, analytically tractable via the central limit theorem; generalisation error decays slowly as α grows
- **Perceptron / Randomised Perceptron** — convergent online rules that approach the Gardner capacity bound at large α
- **Adaline** — gradient descent on MSE; equivalent to the pseudo-inverse rule in the overdetermined regime
- **Teacher-student setup** — labels are generated by a "teacher" perceptron with fixed weights, enabling direct comparison with the theoretical generalisation curves derived from the Gardner / replica analysis

## Classes

### `RandBinary`

A single bipolar random vector of length `L` with ±1 entries.

```cpp
RandBinary(int L);                        // random ±1 vector
RandBinary(const vector<int>& seq);       // from explicit sequence
const vector<int>& getSequence() const;
```

### `RandBinaryDataset`

A dataset of `P` random bipolar input vectors paired with labels.

```cpp
RandBinaryDataset(int P, int L);                            // random ±1 labels
RandBinaryDataset(int P, int L, const Perceptron& teacher); // labels from teacher
const vector<RandBinary>& getData() const;
const vector<int>&        getLabels() const;
int   countErrors(const vector<int>& predictions) const;
void  shuffle();
int   size() const;
```

### `Perceptron`

A single-layer perceptron with Gaussian-initialised weights.

```cpp
Perceptron(int L);                          // random N(0,1) weights
Perceptron(const vector<double>& weights);  // explicit weights (teacher)

double apply(const RandBinary& x) const;    // raw dot product w·x
int    testOnDataset(const RandBinaryDataset& ds) const;  // error count
double calcCost(const RandBinaryDataset& ds) const;       // mean squared error

Perceptron& trainOnDataset(RandBinaryDataset& ds, const string& rule, double variance = 50.0);
```

**Supported `rule` strings:**

| Rule | Description |
|---|---|
| `"Hebb"` | One-shot Hebbian update: `w += (1/√L) y^μ x^μ` |
| `"Perceptron"` | Online perceptron rule, runs until zero training error |
| `"RandomPerceptron"` | Perceptron rule with Gaussian noise on each update |
| `"Adaline"` | Hebbian init → gradient descent on MSE until convergence |

## Compilation

Include all six source files in your build command — no separate compilation step is required:

```bash
g++ -std=c++17 -Ipath/to/gardner-perceptron \
    your_program.cpp \
    path/to/gardner-perceptron/Perceptron.cc \
    path/to/gardner-perceptron/RandBinary.cc \
    path/to/gardner-perceptron/RandBinaryDataset.cc \
    -o your_program
```

The `-I` flag lets `#include "Perceptron.h"` resolve without path prefixes in your source.

## Usage example

```cpp
#include "Perceptron.h"
#include "RandBinaryDataset.h"

int main() {
    int L = 100;  // input dimension
    int P = 50;   // training examples (α = P/L = 0.5)

    // Teacher with random weights generates the labels
    Perceptron teacher(L);
    RandBinaryDataset dataset(P, L, teacher);

    // Train a student perceptron with the Hebb rule
    Perceptron student(L);
    student.trainOnDataset(dataset, "Hebb");

    // Evaluate generalisation error
    int errors = student.testOnDataset(dataset);
    double error_rate = static_cast<double>(errors) / P;
}
```
